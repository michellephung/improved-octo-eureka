# Hangman


###To view static page, 
click on public/index.html

###To Develop:
Run 

	`npm install` 

and wait for modules to download.  

To compile static folder, run:
	
		$ grunt
		
To develop, run:

		$ node app.js
		
you should see: `server is running` in that tab.

In another terminal window, run:

		$ grunt watch
		
Site is hosted at localhost:4000
